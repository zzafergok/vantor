# Vantor

Vantor is a premium, domain-neutral Next.js application template for public, authenticated, and admin surfaces.

It keeps:

- A strong dark interface system and Tailwind theme
- Custom `components/core` primitives
- Form components built on React Hook Form and Zod
- Toast system
- Public, auth, and admin route foundations

It removes:

- Firebase, auth, monitoring, AI, kanban, project-manager, waitlist, and account domains
- Product-specific workflows and business logic
- Domain-specific stores and API routes

## Usage

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
